## About
- A simplified version of Flappy Bird, created as a learning exercise
  - *The current version has no audio or high score persistence*
- Built with C++, [raylib](https://www.raylib.com/) and [CLion](https://www.jetbrains.com/clion/)
- Check out the playable web build [here](https://vidak92.github.io/flappy-bird-cpp-build/)
  - *The current version works only on desktop platforms*
- Assets by [Kenney](https://www.kenney.nl)
